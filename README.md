🧮 Math Kids: Magia & Acero (v5.0)

Aplicación Web Gamificada para el Dominio de las Matemáticas Básicas
Math Kids es una herramienta educativa de alto impacto diseñada para niños en etapa escolar. Transforma la práctica de operaciones aritméticas en una misión épica donde la personalización, el feedback inmediato y el control de la dificultad permiten un aprendizaje adaptativo y sin frustraciones.

✨ Novedades de la Versión 5.0
Esta versión representa una evolución total en la experiencia de usuario y la solidez del sistema:

🛡️ Sistema Anti-Errores (Input Shield): Se implementó una validación mediante Expresiones Regulares (Regex) que bloquea símbolos, puntos y signos negativos. Solo se permiten dígitos, eliminando el bug de bloqueo de casillas.

🎭 Dualidad de Temas (Skins): * Modo Princesa ✨: Colores pastel, diamantes y una estética suave.

Modo Guerrero ⚔️: Tonos azules/verdes, dragones y una estética de batalla.

➕/➖ Selector de Operaciones: Ahora incluye soporte completo para Restas, con un algoritmo inteligente que garantiza que el minuendo sea siempre mayor que el sustraendo (evitando resultados negativos).

🔢 Dificultad Escalar: Selector de 1, 2 o 3 dígitos que reconstruye la interfaz dinámicamente según el nivel de desafío elegido.

🎯 Metas Auto-Definidas: El usuario decide cuántos ejercicios resolver, fomentando la responsabilidad y el establecimiento de objetivos.

🚀 Características Técnicas
Navegación Inteligente: Sistema de auto-enfoque que avanza el cursor al escribir y lo retrocede al borrar (Backspace).

Teclado Optimizado: Uso de type="tel" para forzar el teclado numérico en dispositivos móviles, mejorando la ergonomía.

Arquitectura Vanilla: Construido exclusivamente con HTML5, CSS3 y JavaScript ES6+. Sin librerías externas para una carga instantánea (menos de 1s).

Responsive & Mobile First: Interfaz diseñada específicamente para el uso táctil en tablets y smartphones.

🎮 Instrucciones de Uso
Configuración: Al abrir la app, selecciona tu personaje, la operación (Suma o Resta) y la cantidad de dígitos.

Meta: Escribe tu nombre y cuántos retos quieres superar hoy.

Resolución: Escribe el resultado empezando por las unidades. El cursor saltará solo a las decenas y centenas.

Confirmación: Presiona el botón "Comprobar" o simplemente la tecla Enter en tu teclado.

Desafío: ¡Cuidado con la nube o el dragón! Si te equivocas varias veces, se acercarán a tu personaje.

🎓 Valor Pedagógico
Esta aplicación ha sido diseñada siguiendo principios de Micro-learning y Gamificación, ayudando a desarrollar:

Agilidad Mental: Resolución rápida de problemas bajo presión lúdica.

Concepto Posicional: Refuerza la comprensión de Unidades, Decenas y Centenas mediante casillas separadas.

Autonomía: El niño autogestiona su progreso y meta diaria.

Nota: Esta aplicación es de código abierto y puede ser utilizada por docentes y padres para complementar la enseñanza en casa o en el aula.
